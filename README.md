# BITMAN

A retro-style platformer game built with SolidJS and PixiJS. Control Bitman and avoid falling bombs!

## Features

- Difficulty selection (Easy: 2, Medium: 3, Hard: 5 bombs)
- High score tracking per difficulty (saved in localStorage)
- Game over screen with click-to-continue

## How to Play

1. Select a difficulty level
2. Use arrow keys or A/D to move left and right
3. Avoid falling bombs
4. Score points by letting bombs explode on the platform

## Development

```bash
npm install
npm dev      # Start development server
npm run build # Build for production
```
